# Code of Conduct

## Our Standards

This is a professional engineering project. Contributors and users are expected to:

- Be respectful and professional in all interactions
- Provide constructive feedback on code and design
- Accept feedback gracefully
- Focus discussion on technical merit

Unacceptable behavior includes:
- Harassment, discrimination, or personal attacks
- Using this project for unauthorized access to systems or content
- Sharing platform credentials, auth tokens, or DRM keys in public issue comments
- Distributing this tool to unauthorized parties

## Legal Compliance

Users of this tool are solely responsible for ensuring their use complies with:
- Applicable laws (DMCA, CFAA, GDPR, IT Act 2000, and equivalents)
- Platform Terms of Service
- Contractual obligations to content owners

## Reporting

Code of conduct violations: open a GitHub issue or contact via GitHub profile

Security vulnerabilities: see [SECURITY.md](./SECURITY.md)
